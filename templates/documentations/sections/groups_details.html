Groupe Viewer (Niveau de base)

Accès en lecture seule à toutes les données

Permissions détaillées:

Consultation des projets
Consultation des tests
Consultation des exécutions
Consultation des résultats de tests
Consultation des tags
Accès au dashboard et statistiques
Consultation des configurations CI
Consultation des features projets
Restrictions:
Aucune création/modification
Pas d'import de résultats
Pas de gestion des tags
Pas d'accès à l'administration

Groupe Manager (Viewer + permissions supplémentaires)

Gestion des tests et tags, lecture seule des projets

Hérite de toutes les permissions Viewer

Permissions supplémentaires:

Création/modification des tests
Import/modification des résultats
Gestion des tags (couleurs, etc.)
Modification des statuts de tests
Ajout/modification des commentaires
Configuration des features projets
Gestion des configurations CI
Gestion des clés API
Restrictions maintenues:
Pas de gestion des projets
Pas d'accès à l'administration

Groupe Admin (Manager + permissions supplémentaires)

Accès complet à toutes les fonctionnalités

Hérite de toutes les permissions Manager (donc aussi Viewer)

Permissions supplémentaires exclusives:

Création/modification/suppression des projets
Accès au backend
Gestion des utilisateurs et groupes
Configuration système complète
