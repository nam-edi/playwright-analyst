Information importante

• Les utilisateurs peuvent appartenir à plusieurs groupes à la fois.
• Les permissions sont cumulatives : si un utilisateur est dans plusieurs groupes, il aura toutes les permissions de ces groupes.
• Les superutilisateurs Django ont automatiquement toutes les permissions, indépendamment de leurs groupes.
