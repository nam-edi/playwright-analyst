Instructions d'Utilisation des Comptes de Test

Comment tester

  1. 1 Déconnectez-vous de votre compte actuel
  2. 2 Connectez-vous avec l'un des comptes de test ci-dessus
  3. 3 Explorez l'interface pour voir les différences d'accès
  4. 4 Testez les fonctionnalités autorisées/interdites

Points d'attention

  • Ces comptes contiennent des données de test, pas vos données réelles
  • Les modifications sur ces comptes peuvent être réinitialisées
  • N'utilisez pas ces identifiants en production
  • Revenez à votre compte principal après les tests
